## üîó Connecting the Pipeline

### Content Generation Flow

1. **Trigger:** Daily schedule or manual request
2. **Generation:** CoFounder Agent creates content
3. **Storage:** Saves to Strapi via API
4. **Publishing:** Content becomes available
5. **Distribution:** Automatically syndicated to web/social

### Required Strapi API Tokens

Create in Strapi Admin:

```
Settings ‚Üí API Tokens ‚Üí Create New Token

- Name: Content Pipeline
- Description: For automated content generation
- Permissions: Full access to blog posts
- Expiration: None (or yearly)
```

Copy the token and add to your environment variables:

```
STRAPI_API_TOKEN=<your-token>
STRAPI_API_URL=https://glad-labs-strapi-v5-backend-production.up.railway.app
```
