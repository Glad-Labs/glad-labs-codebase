# React (Oversight Hub) Instructions

- **Project Root:** `web/oversight-hub`
- **Framework:** React (Create React App)
- **Key Libraries:** `react-router-dom`, `firebase` (for Firestore), `@chatscope/chat-ui-kit-react`, `zustand`.
- **Styling:** Tailwind CSS.
- **Startup:** The app is started with `npm start --workspace=oversight-hub`.
- **Data Fetching:** Data is fetched from a Google Cloud Firestore database. Configuration is in `src/firebaseConfig.js`.
