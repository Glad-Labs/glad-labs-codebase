# Next.js (Public Site) Instructions

- **Project Root:** `web/public-site`
- **Framework:** Next.js
- **Data Source:** Fetches content from the Strapi v5 REST API.
- **API Interaction:** All API calls are handled in `lib/api.js`.
- **Rendering:** The site uses Static Site Generation (SSG) with Incremental Static Regeneration (ISR).
- **Styling:** Tailwind CSS.
