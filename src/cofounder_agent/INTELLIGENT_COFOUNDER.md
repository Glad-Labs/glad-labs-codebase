### Basic Usage

```python
import asyncio
from cofounder_agent.intelligent_cofounder import IntelligentCoFounder

async def main():
    # Initialize AI Co-Founder
    cofounder = IntelligentCoFounder("Your Business Name")
    await cofounder.initialize()

    # Start conversation
    response = await cofounder.chat("How is my business doing overall?")
    print(response)

asyncio.run(main())
```
